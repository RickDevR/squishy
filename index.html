About Squishy Universe

This is a fully animated squishy playground with 30 unique shapes, colors, and satisfying sounds. The first 10 squishies use your own audio files, and the remaining 20 use generated squishy tones built with the Web Audio API.

Everything runs from a single folder with no subfolders, designed for a clean, modern, sleek UI and easy expansion later (shops, upgrades, saving, etc.).
